# modern-react-boilerplate
Easy boilerplate for React 16, Webpack 4, React Router 4

## Features

* React 16
* Webpack 4
* Babel
* Staging ES Next Features
* Jest and Enzyme

## Installation
* `git clone https://github.com/Buul/react-easy-boilerplate.git`
* cd react-easy-boilerplate
* npm install

### run
npm start
